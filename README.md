# Customer Feedback Sentiment Analysis

## Project Overview
This project analyzes customer feedback using Natural Language Processing (NLP) and Machine Learning to predict sentiment.

## Technologies Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Dataset
Twitter Sentiment Dataset

## Features
- Data Cleaning
- Sentiment Distribution Visualization
- Text Vectorization (TF-IDF)
- Sentiment Prediction using Machine Learning

## Conclusion
The model predicts customer sentiment from text feedback with good accuracy.
